About Me

Welcome to Project 1 of INFO H517- Data Visualization. I am Sindhuja Madishetty. Currently a graduate student in Applied Data Science Specialization at Indiana University Purdue University Indianapolis. This project has been done as a part of Visualization Design, Analysis and Evaluation course.

This project is a replication of the famous cholera map by John Snow, which played a key role in stopping the spread of cholera epidemic in Soho, London in 1854. This map helped in identifying that the major reason for the spread of cholera in London in 1854 is the pump located on the Broad street.

Visualization Details

I have implemented this project using D3.js, HTML and CSS. Dataset consists of 4 files with informations like locations of streets, locations of pumps, number of deaths on each day and file with locations, age and gender of each death. The names of the major streets, locations of brewery and work house have been approximated on the map. User have the option to select to visualize deaths on map based on age and gender with the help of buttons. There is a timeline chart representing number of deaths on each day and also a grouped bar chart representing number of deaths by age group and gender. User have the option to click a particular data point on the timeline chart, so that deaths recorded on that day will be updated on the map and grouped bar charts. There is also a provision to select a particular period of time beneath the timeline chart, so that deaths recorded in that time period will be reflected on the timeline chart.
